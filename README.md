# AI Financial Knowledge System
An AI-powered financial education platform that helps users understand topics like Stocks, Crypto, Banking, and Mutual Funds in a beginner-friendly and structured format.

### The System Does:
1️⃣ Fetches verified information from the internet  
2️⃣ Uses Groq LLM to generate detailed explanations  
3️⃣ Provides curated learning articles  
4️⃣ Provides topic-based YouTube learning videos  
5️⃣ Explains benefits, risks & roadmap clearly  
6️⃣ Displays output in beautiful FinTech UI
